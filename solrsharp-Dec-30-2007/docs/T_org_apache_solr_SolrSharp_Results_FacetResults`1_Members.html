FacetResults Members

FacetResults is an abstraction from the details of determining the results from a query using facet parameters, and returns the results in an object-based structure. FacetResults requires an implementation utilizing generics. This enables the base class to build facet result objects directly. Using generics, the base class populates Facet results. Inheriting classes need only define how a Facet object is initialized. The base class executes those inherited methods as part of a chain of calls from SearchResults.ExecuteSearch. This class is abstract (MustInherit in Visual Basic)abstractMustInherit and so cannot be instantiated.
