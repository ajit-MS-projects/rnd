SolrSharp - C# API to Apache Solr
org.apache.solr.SolrSharp.Query.Parameters Namespace
See Also Example Hierarchy

Classes

  Class Description
IdListQueryParameter Specialized QueryParameter used to build a query for a list of IDs on a solr index fieldname
IdListQueryParameterCollection Inheritance of the QueryParameterCollection that facilitates queries of specific IDs from a list.
QueryParameter A field/value pair used as a building block for a Query object. One or QueryParameter objects can be grouped using a QueryParameterCollection, and subsequently added to a Query object for used with a QueryBuilder object. This object may be used stand alone or part of a group of a larger collection of QueryParameter objects in constructing a complex search query.
QueryParameterCollection A collection of QueryParameter objects, unified to be applied in a specific grouping for a search request.
RangeQueryParameter Specialized QueryParameter object that constructs a range-based query with start and end values
StartsWithQueryParameter Specialized QueryParameter object that constructs a logical Lucene-formatted query, similar to .Net String.StartsWith()
